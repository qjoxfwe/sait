Задание для обучения текстовым процессорам №4

Задания для личного развития

Практические занятия предполагают работу, связанную с обработкой текстовой и числовой информации; с применением мультимедийных технологий обработки и представления информации; с обработкой экономической и статистической информации с помощью соответствующих пакетов прикладных программ.

Структура практических занятий включает в себя:

  • теоретические вопросы по изучаемым темам
  • практические задания для решения с помощью программных средств

СОЗДАНИЕ И РЕДАКТИРОВАНИЕ ТАБЛИЦ. СОРТИРОВКА ТАБЛИЦ. ВЫЧИСЛЕНИЯ В ТАБЛИЦАХ. ПРЕОБРАЗОВАНИЕ ТЕКСТА В ТАБЛИЦУ

Цели: познакомиться с текстовым процессором, научиться создавать и редактировать таблицы.

Теоретические вопросы

  1. Способы создания таблиц.
  2. Форматирование таблиц
  3. Сортировка таблиц.
  4. Вычисления в таблицах.
  5. Преобразование текста в таблицу.

Задание 4. Создайте таблицу:

Итоговые значения в последнем столбце подсчитайте, используя команду Таблица – Формула. В диалоговом окне будет предложена формула =SUM(LEFT), выполняющая вычисление суммы чисел в ячейках слева.

Просмотрите список предлагаемых формул.

При изменении данных в ячейках сумма автоматически не пересчитывается. Замените количество стульев в сентябре: вместо 112 введите 150. Установите курсор на итоговую сумму стульев (она выделится затенением) и нажмите F9. Результат обновится. Другой способ: вызвать контекстную команду Обновить поле.
